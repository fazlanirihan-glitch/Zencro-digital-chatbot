# ZenCro AI Platform

![ZenCro AI Platform](https://placehold.co/1200x400/0f172a/3b82f6?text=ZenCro+AI+Platform)

**Version:** 1.0.0 (Production Release)

ZenCro AI is an enterprise-grade, multi-tenant AI Platform built to seamlessly capture leads, answer complex inquiries via Retrieval-Augmented Generation (RAG), and manage multi-channel conversations.

## 🌟 Key Features

### 1. **Premium White-Label Chat Interface**
- Dynamic Glassmorphism UI powered by Framer Motion.
- Fully customizable Hex Colors, Avatars, and Welcome Messages.
- Multi-step conversational Lead Capture cards.
- Real-time token streaming and dynamic AI suggestions.

### 2. **Multi-Tenant SaaS Engine**
- Secure RAG isolation by `company_id`.
- Tenant Management dashboard for onboarding new clients.
- Automated white-label Next.js bundle generation.

### 3. **Knowledge Engine (RAG)**
- Drag-and-drop parsing for `.pdf`, `.docx`, `.md`, and `.txt` files.
- Real-time token tracking and semantic chunking.

### 4. **Multi-Channel Ready**
- Abstracted interfaces for Voice Synthesis, WhatsApp (Meta Webhooks), and Email alerts.

## 🛠️ Tech Stack
- **Frontend**: Next.js 15, React 19, Tailwind CSS v4, Framer Motion, Lucide Icons.
- **Backend**: Python 3.11, FastAPI, Supabase (PostgreSQL), Langchain.
- **Infrastructure**: Docker, Redis.
